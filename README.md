# Nathan Onduma Quarto Website

Starter Quarto personal website that merges the existing CV with RPubs portfolio work.

## Preview locally

```bash
quarto preview
```

## Render

```bash
quarto render
```

## Publish to GitHub Pages

```bash
quarto publish gh-pages
```

Before publishing, update `site-url` in `_quarto.yml` to your final website URL.
